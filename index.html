Knowledge Transfer for Robust GeoAI Across Space, Sensors and Time

Principal Investigator: Saurabh Prasad

Co-PI: Melba Crawford

About This Research Initiative

This initiative develops novel deep learning approaches to enable robust artificial intelligence for geospatial analysis across different sensors, time periods, and geographical locations. Through domain generalization, foundation models, and active learning strategies, we advance the state of AI for Earth observation and remote sensing applications. Our work addresses critical challenges in model transferability, annotation efficiency, and multi-modal data integration.

Research Contributions

Sensor Agnostic Domain Generalization Framework

A novel framework that leverages geospatial foundation models for semantic segmentation across different sensors through synergistic pseudo-labeling and generative learning. Addresses annotation scarcity and sensor variability challenges in remote sensing.

Domain Generalization Foundation Models Semantic Segmentation Pseudo-Labeling
Authors: Anan Yaghmour, Melba Crawford, Saurabh Prasad
View Code Read Paper

Efficient Vision Foundation Models for Multispectral Analysis

A self-supervised vision foundation model designed specifically for multispectral remote sensing imagery. Enables efficient fine-tuning with only 5.2% trainable parameters while maintaining high generalization across downstream tasks and datasets.

Foundation Models Self-Supervised Learning Multispectral Imagery MAE
Authors: Wei Liu, Saurabh Prasad, Melba Crawford
View Code Read Paper

Attention Guided Semisupervised Generative Transfer Learning

A domain adaptation technique for hyperspectral image analysis that combines attention mechanisms with generative models. Addresses domain shift challenges by leveraging limited labeled data from target domains while maintaining robustness across sensors and geographic locations.

Domain Adaptation Transfer Learning Hyperspectral Imagery Attention Mechanisms
Authors: Anan Yaghmour, Saurabh Prasad, Melba Crawford
View Code Read Paper

Hierarchical Spectral Vision Transformer Architecture

A novel hierarchical vision transformer architecture specifically designed for hyperspectral image classification. Integrates CNN mixers and self-attention mechanisms to effectively process spectral information and spatial features in hyperspectral data.

Vision Transformers Hyperspectral Classification Self-Attention Deep Learning
Authors: Wei Liu, Saurabh Prasad, Melba Crawford

Research Focus Areas

🌍 Domain Generalization

Developing models that transfer across different sensors, illumination conditions, and geographic regions without extensive retraining.

🤖 Geospatial Foundation Models

Creating large-scale self-supervised pre-trained models optimized for remote sensing and Earth observation tasks.

📡 Multi-Sensor Integration

Building robust approaches that work across multispectral, hyperspectral, and other sensor modalities simultaneously.

⏰ Temporal Analysis

Integrating temporal dynamics to capture Earth system changes over time while maintaining model robustness.
